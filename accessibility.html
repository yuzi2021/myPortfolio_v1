Accessibility Statement

Last updated: May 2026  ·  Sarah Sandström Portfolio

This site aims to meet WCAG 2.1 Level AA. This statement is honest about what is currently implemented and what is still being improved.

About this site

This is a personal portfolio website belonging to Sarah Sandström, hosted on GitHub Pages. The site is maintained by one person and is actively being improved for accessibility as pages are updated.

Standard and scope

The target standard is the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA. This is also the standard referenced in the Finnish Act on the Provision of Digital Services (Laki digitaalisten palvelujen tarjoamisesta, 306/2019), although that Act applies to public sector services rather than personal portfolio sites.

Known issues

Footer year: Some pages show 2024 in the footer. This is being corrected as pages are updated.

Project image alt text: Some project card images may have incomplete or missing alternative text. This is being audited.

Improvement approach

Accessibility improvements are being made incrementally as each page of the portfolio is updated. The priority order is: navigation and structure, images and alt text, colour contrast, keyboard and focus, then page-specific components.

This statement is updated each time a page is revised. The date at the top reflects the last review.

Feedback and contact

If you encounter an accessibility barrier on this site, please contact sarahyuzisandstrom@gmail.com. Feedback is genuinely welcome and will be acted on.
