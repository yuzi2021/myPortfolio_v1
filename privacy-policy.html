Privacy Policy

Last updated: May 2026  ·  Sarah Sandström Portfolio

This site does not use cookies, does not collect personal data, and does not track visitors. This policy explains what that means in practice.

What this site is

This is a personal portfolio website belonging to Sarah Sandström. Its purpose is to present professional experience, projects, and contact information. It is hosted on GitHub Pages.

Data collected by this site

This site does not collect, store, or process any personal data. There are no contact forms, no newsletter sign-ups, no user accounts, and no analytics scripts. No cookies are set by this site.

GitHub Pages hosting

This site is hosted by GitHub Pages (GitHub, Inc.). GitHub may collect server log data including IP addresses as part of standard infrastructure operation. This is outside the control of the site owner. GitHub's privacy practices are documented at docs.github.com.

Email contact

If you contact the site owner by email at sarahyuzisandstrom@gmail.com, your email address and the content of your message will be received and processed by the site owner for the purpose of responding to your enquiry. Email is not sent through this website — it is handled directly by your email client.

External links

This site contains links to external services including LinkedIn, GitHub, and project documentation. Following these links takes you away from this site and into those services' own privacy policies and data practices, which are outside the control of the site owner.

FridgeReady project

The FridgeReady project is documented on this portfolio but is a separate application with its own privacy policy and data controller. Any personal data processed by FridgeReady is governed by FridgeReady's own documentation, not by this portfolio's privacy policy.

Your rights

Because this site does not collect personal data, there is nothing to access, correct, or delete. If you have a question about data that may have been collected by GitHub Pages infrastructure, please refer to GitHub's privacy documentation above.

Contact

Questions about this privacy policy can be directed to sarahyuzisandstrom@gmail.com.

Changes to this policy

If this site adds features that involve data collection in the future, this policy will be updated before those features go live. The date at the top of this page reflects when the policy was last reviewed.
